2、端口PORT

==端口表示计算机上的一个程序的进程==

  • 不同的进程有不同的端口号！用来区分软件

  • 被规定的数量为0~65535

  • TCP/UDP：都有65535个端口号，其中UDP和TCP可以使用相同的端口号，==但是在同一个协议之下不能使用同一个端口号==

  • 端口分类：

    1. 公有端口：0~1023

      • HTTP：80
      • HTTPS：443
      • FTP:21
      • Telent:23

    2. 程序注册端口：1024~49151，分配用户或者程序

      • Tomcat：8080
      • MySql：3306
      • Oracle：1521

    3. 程序注册端口：49152~65535

    4. ```java
      package NetProgram;

      import java.net.InetAddress;
      import java.net.InetSocketAddress;
      import java.net.UnknownHostException;

      public class Demo01 {

      public static void main(String[] args) throws UnknownHostException {
    InetAddress ia = InetAddress.getByName("www.baidu.com");//关键代码
    System.out.println(ia);
    System.out.println(ia.getHostAddress());

    InetSocketAddress isa = new InetSocketAddress("127.0.0.1", 3306);//关键代码
    System.out.println(isa.getAddress());
    System.out.println(isa.getHostName());
    System.out.println(isa.getPort());
}

      }

      输出：
      www.baidu.com/14.215.177.38
      14.215.177.38
      /127.0.0.1
      127.0.0.1
      3306
